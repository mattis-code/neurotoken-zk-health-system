# Contributing Guide

## Structure
- docs/ — documentation
- sdk/ — language SDKs
- examples/ — demo integrations
- tests/ — unit & integration tests

## Setup
Clone repo:
git clone https://github.com/mattis-code/neurotoken-zk-health-system

## Workflow
1. Create feature branch
2. Commit with conventional commits
3. Ensure tests pass
4. Open PR
